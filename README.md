# Tabletop Role-Playing Game? OMG!

[ttrpgomg.com](https://ttrpgomg.com)

## Getting Started

```sh
# install dependencies
npm install

# run in development mode
npm start

# build the site for production
npm run build

# lint css and js
npm run lint

# lint html (run after site is built)
npm run lint:html

# run js tests
npm run test
```
